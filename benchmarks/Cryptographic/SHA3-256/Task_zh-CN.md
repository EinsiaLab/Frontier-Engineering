# SHA3-256 (Secure Hash Algorithm 3 256-bit)
SHA3-256 是 NIST 通过公开密码学竞赛选出的最新一代散列函数标准（FIPS 202）。其底层采用 Keccak 算法，同样输出 256 比特的摘要。

SHA3-256 采用与 SHA-2 截然不同的海绵构造（Sponge Construction）。依赖于内部一个 1600 比特的巨大状态空间（State），主要分为两个阶段：
- **吸收阶段（Absorbing）**：将输入消息按照设定的比特率（Bitrate）分块，与状态空间的前半部分进行异或，并对整个状态空间应用 Keccak-f 核心置换函数。
- **挤出阶段（Squeezing）**：在吸收完所有消息后，直接从状态空间的前半部分提取所需长度（256 比特）的哈希输出。
