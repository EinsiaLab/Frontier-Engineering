# SHA-256 (Secure Hash Algorithm 256-bit) 算法加速
SHA-256 是由美国国家标准与技术研究院（NIST）发布的 SHA-2 散列函数家族成员，能够为任意长度的输入数据计算出固定长度为 256 比特（32 字节）的消息摘要（Message Digest）。
SHA-256 基于传统的 Merkle-Damgård 构造。首先对输入消息进行填充（Padding），使其位长度模 512 与 448 同余，并附加 64 比特的原始消息长度信息，使得最终长度严格为 512 比特的整数倍。随后，通过初始哈希值和由复杂逻辑运算（位操作、模加等）构成的压缩函数，以 512 比特为一个区块逐块迭代处理数据，最终输出 256 比特的散列值。
