# 直流最优潮流 — IEEE 14 节点系统

## 概述

在满足直流潮流方程和线路热稳定约束的前提下，最小化 14 节点电力系统的发电成本。
这是经典的**直流最优潮流（DC-OPF）**问题，使用 IEEE 14 节点标准测试系统。

| 属性 | 值 |
|------|----|
| 领域 | 电力系统 |
| 任务形式 | 含物理约束的连续优化 |
| 难度 | 3 / 5 — 需要建模直流潮流；支路 2→3 约束具有约束性 |
| 依赖 | `numpy`, `scipy` |
| 人类最优成本 | **7 892.76 $/h**（SLSQP 求解 DC-OPF 最优）|
| 基线成本 | 9 154.77 $/h（均等调度）→ 得分 **0.862** |

## 问题描述

给定一个含 14 个节点、5 台发电机、20 条支路的电力网络，找到最优的有功调度方案
**Pg = [Pg₁, Pg₂, Pg₃, Pg₄, Pg₅]**（MW），使总发电成本最小：

```
cost = Σᵢ (aᵢ · Pgᵢ² + bᵢ · Pgᵢ)   [$/h]
```

满足约束：

1. **功率平衡**：`sum(Pg) = 259 MW`（系统总负荷）
2. **发电机限值**：`Pmin_i ≤ Pg_i ≤ Pmax_i`
3. **直流线路潮流约束**：`|Pᵢⱼ| ≤ rate_aᵢⱼ`（通过直流潮流 `B·θ = P_net` 计算）

支路 2→3 的热稳定限额（36 MW）在最优点处**具有约束性**，导致朴素的经济顺序调度会违反此约束，使问题非平凡。

## 评分

```
score = min(1.0, HUMAN_BEST_COST / solution_cost)
      = min(1.0, 7892.76 / cost)
```

越大越好；1.0 表示达到或超越 DC-OPF 最优。
不可行解（功率不平衡、超越发电机限值或违反线路热稳定约束）得分为 0。
