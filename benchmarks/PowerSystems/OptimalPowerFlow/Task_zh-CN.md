# 任务：直流最优潮流 — IEEE 14 节点

## 问题

最小化 IEEE 14 节点电力系统的总发电成本（$/h）。

**输入**（`instance` 字典）：
- `base_mva`（float）：100 MVA 系统基准
- `generators`：5 个发电机的列表，每个包含：
  `{bus, pmin, pmax, cost_a, cost_b}` — 节点编号、MW 限值、二次成本系数
- `buses`：14 个节点的列表，每个包含：
  `{id, pd}` — 节点编号和有功负荷（MW）
- `branches`：20 条支路的列表，每个包含：
  `{from_bus, to_bus, x, rate_a}` — 连接关系、标幺电抗、热稳定限额（MW）

**输出**：长度为 5 的 `list[float]`
发电机有功出力 `[Pg1, Pg2, Pg3, Pg4, Pg5]`，单位为 MW。

## 约束

| 约束 | 公式 | 容差 |
|------|------|------|
| 功率平衡 | `\|sum(Pg) - 259\| ≤ 0.5 MW` | 硬约束 |
| 发电机限值 | `pmin_i ≤ Pg_i ≤ pmax_i` | 硬约束 |
| 直流线路潮流 | `\|P_ij\| ≤ rate_a_ij`，通过 `B·θ = P_net/base_mva` 计算 | 硬约束 |

任何约束违反 → `valid = 0`，`score = 0`。

## 评分

```
score = min(1.0, 7892.76 / cost)
```

其中 `cost = sum(cost_a[i] * Pg[i]^2 + cost_b[i] * Pg[i])` $/h。

- 人类最优（DC-OPF 最优）：**7 892.76 $/h → 得分 1.000**
- 基线（均等调度）：        **9 154.77 $/h → 得分 0.862**

## 关键难点

支路 2→3 的**热稳定限额为 36 MW**（在最优点处具有约束性）。
朴素的按成本顺序调度（将廉价电力全部从 1 或 2 号节点出力）会导致该线路过载，
使方案不可行。最优调度需要在成本最小化与线路 2→3 的网络约束之间取得平衡。

## 提示

- 直流潮流方程：`B · θ = P_net / base_mva`
  其中 `B[i,i] = sum(1/x_ij)`，`B[i,j] = -1/x_ij`
- 节点 1（索引 0）为平衡节点：`θ[0] = 0`（求解时去除对应行和列）
- 支路 i→j 的有功潮流：`P_ij = (θ_i - θ_j) / x_ij * base_mva`
- 可使用 `scipy.optimize.minimize`（SLSQP 方法）求解该含约束二次规划问题
