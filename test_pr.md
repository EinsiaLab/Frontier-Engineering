OpenAI 报告了一个“实验室在环（lab-in-the-loop）”的结果：他们把 GPT-5 连接到 Ginkgo Bioworks 的云端机器人实验室，用来优化 无细胞蛋白质合成（CFPS）——一种不需要培养活细胞、在体外体系里生产蛋白质的方法。
他们做了什么
搭建闭环自主流程：GPT-5 提出一批实验方案 → 机器人在 384 孔板上执行 → 数据回传 → GPT-5 分析并设计下一轮。
共进行了 6 轮迭代，测试了 36,000+ 种不同的 CFPS 反应配方，覆盖 580 块自动化板。
核心结果
在获得工具权限（电脑 + 浏览器 + 论文）后，GPT-5 用 3 轮迭代就把成本推到了新的低成本 SOTA，最终宣称：
蛋白生产成本降低 40%
试剂成本效率提升 57%
同时找到了在自动化实验室“现实条件”下更稳健的新型反应配方（例如微孔板体系常见的低氧环境）。
他们学到了什么 / 为什么有效
大幅提升主要来自：找到一些成分组合上的协同，这些组合在该配置下人类此前没系统探索过。
高通量微孔板条件与台式试管实验不同（供氧、混合、几何形态都不同），所以“经典配方”并不能直接迁移。
一些相对便宜的组分（缓冲体系、能量再生组分、多胺等）对结果产生了超出其成本的影响。
由于成本主要被裂解液（lysate）和 DNA 主导，提高**单位昂贵输入对应的产量（yield）**是最有杠杆的策略。
局限与注意点
只在 一种蛋白（sfGFP） 和 一种 CFPS 系统上验证；是否能泛化到其他蛋白/体系尚未证明。
结果可能对供氧/几何结构以及规模变化敏感。
在试剂处理、协议细节等方面仍需要人类监督。
下一步
把同样的闭环自主优化方法扩展到其他生物工作流，同时在其 Preparedness Framework 下评估潜在生物安全影响并做风险缓解。
